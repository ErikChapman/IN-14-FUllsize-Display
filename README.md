# Overview

This repository showcases a detailed project incorporating various components used for full-size display applications.

# Features

- **Microcontroller**: ESP32-WROOM-32U (16MB)
- **RGB LEDs**: 6 multi-colored LED setup for vibrant display effects.
- **Display**: TFT display for high-quality graphics and interactions.
- **Sensors**:
  - BMP280 sensor for atmospheric pressure and temperature measurements.
  - SHT30-DIS-B2.5KS sensor for accurate humidity readings.
- **Indicators**: IN-14 indicators that enhance the visual appeal of this project.
