# IN-14 FUllsize Display

## Introduction
This project showcases a full-size display utilizing the TFT display along with additional sensors such as the BMP280 and SHT30-DIS-B2.5KS for enhanced functionality.

## Features
- **Full-Color TFT Display**: High-resolution display for vivid graphics and clear visibility.
- **BMP280 Sensor**: Accurate temperature and pressure readings for environmental monitoring.
- **SHT30-DIS-B2.5KS Sensor**: Provides precise humidity and temperature measurements, ideal for climate control applications.
